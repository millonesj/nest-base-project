# **NEST-BASE-PROJECT**

# Resumen
This is a nestjs base project for building APIs. It contains the base configuration and hex architecture folder layout.

# Technology Stack

- Nest
- Typescript
- Swagger

# Commands
## Instalation

steps:

>1. Clone repository:  
    `git clone git@github.com:millonesj/nest-base-project.git`  

>2. Install dependencies:  
`yarn install`
## Execution  
>Execute in development enviroment:  
`yarn start:dev`

## Publish
>Compile Typescript to Javascript. The files are generated in  `dist/` folder.  
`yarn build`

## Test
>`yarn test`
